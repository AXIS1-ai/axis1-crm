# CRM Definitivo AXIS 1

Versão unificada com:

- Leads e clientes no mesmo sistema
- Follow-up automático para leads
- Cobrança inteligente para clientes ativos
- Controle de pagamento mensal ou 2x
- Botões Pago / Pago 1 / Pago 2
- Receita recebida no mês
- Meta mensal e valor faltante
- Receita ativa mensal
- Clientes atrasados em destaque
- Próximos vencimentos nos alertas
- Histórico de pagamentos por cliente
- Backup manual e automático
- Migração automática de versões antigas

## Antes de subir no GitHub
1. Abra o CRM atual.
2. Exporte backup.
3. Suba os 4 arquivos desta versão.
4. Teste em aba anônima.
5. Importe backup caso necessário.

Arquivos:
- index.html
- style.css
- script.js
- README.md
